# Assets

Place ici :
- `icon.png` — 1024×1024 PNG opaque (App Store icon)
- `adaptive-icon.png` — 1024×1024 transparent (Android adaptive foreground)
- `splash.png` — 1242×2436 PNG (splash screen, fond #1A1230)
- `notification-icon.png` — 96×96 PNG blanc sur fond transparent (Android requires alpha-only)

Avant la première EAS build, génère ces 4 PNG depuis le logo « baba IAssou3 ». Un service gratuit comme [hotpot.ai/icon-resizer](https://hotpot.ai) ou `npx expo-asset-utils` peut tout exporter depuis un seul 1024×1024.
